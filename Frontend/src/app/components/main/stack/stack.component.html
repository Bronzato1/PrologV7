Stack

My preferred stack consists of Angular for the frontend, providing a dynamic and responsive user interface. For the backend, I rely on the exceptional capabilities and versatility offered by .NET, either with Azure Functions for serverless computing or the traditional Core MVC. SQL Server handles data storage, ensuring efficient and reliable data management. Code is version-controlled and collaboratively managed using GitHub, facilitating seamless team collaboration and integration.
